## 二、输入方式：一次性完整输入，绝对非交互、无动态选择
这是比赛的核心评测规则，没有任何变通空间，必须严格遵守：
1.  **单轮一次性全量输入**
    官方离线评测的标准流程是：对每一道题，模型只会收到**固定的2份内容**，且是一次性全部喂入，没有第二次输入机会：
    - 你提交的**完整cheatsheet全文**（不是部分、不是节选）
    - 当前待解答的单道题目（等式1、等式2，判断前者是否蕴含后者）
    模型必须在单轮推理中，直接输出True/False，无多轮对话、无二次交互、无补充输入的可能。

2.  **绝对禁止交互性选择/动态调用部分内容**
    任何让模型“按需选择cheatsheet片段”的设计都是无效的，甚至会大幅降低正确率：
    - 比如写“请先看目录，根据题目类型选择对应的规则章节”，完全不符合单轮全量输入的逻辑，模型不会“回头找、按需挑”，反而会因为内容结构混乱误解规则；
    - 比如想搞多轮交互，第一轮让模型判断题目类型，第二轮给对应规则，这完全违反评测的单轮、无交互要求，官方环境根本不支持。

3.  **全程固定，一题不改**
    你只能提交**唯一一份cheatsheet**，官方会用这完全相同的一份文本，评测所有测试题。你不能针对简单题用短版本、难题用长版本，不能针对不同题目调整任何一个字符，完全不存在“选择性输出部分内容”的可能。

## 三、参赛必避的3个常见误区
1.  **误区：靠花哨的prompt技巧就能取胜**
    纠正：比赛的核心是**数学规则的提炼能力**，不是prompt话术。空泛的角色设定、情绪引导、格式技巧几乎没用，只有准确、精炼、可执行的等式推理规则，才能稳定提升正确率，这也是官方把它叫做“cheatsheet（备忘单）”而非“prompt”的核心原因。

2.  **误区：搞复杂的分块、目录、调用逻辑，能让模型更好地用规则**
    纠正：10KB的硬上限根本容不下复杂的调用逻辑，且单轮全量输入的场景下，模型对文本的注意力是前密后疏的。最优方案是**按规则的高频优先级排序，把最常用、最核心的规则放在最前面**，平铺直叙，不要搞嵌套、目录、跳转逻辑。

3.  **误区：测试的时候可以针对单题改cheatsheet，看效果**
    纠正：本地测试必须完全模拟官方流程——**每一道题都用完全相同的完整cheatsheet**，不能针对单题调整内容。只有这样测出来的正确率，才和官方最终评测的结果一致，否则会出现“本地测试正确率很高，提交后分数暴跌”的情况。

## 一、入门级示例（10道，适合新手理解核心逻辑）
### True示例（5道）
1.  前提：`x * x = x`（幂等律）
    结论：`(x * x) * (x * x) = x`
    正确答案：True
    解析：根据前提，`x*x=x`，可将结论左边的`(x*x)`全部替换为`x`，左边变为`x*x`，再根据前提替换为`x`，与右边相等，必然成立。

2.  前提：`x * y = y * x`（交换律）
    结论：`(x * y) * z = z * (x * y)`
    正确答案：True
    解析：前提规定任意两个元素的运算满足交换律，将`(x*y)`看作一个整体，它和`z`的运算必然满足交换律，因此结论成立。

3.  前提：`x * (y * z) = (x * y) * z`（结合律）
    结论：`x * (y * (z * w)) = ((x * y) * z) * w`
    正确答案：True
    解析：根据结合律，可逐步调整括号位置，左边先把`y*(z*w)`替换为`(y*z)*w`，再把`x*((y*z)*w)`替换为`((x*y)*z)*w`，与右边完全一致。

4.  前提：`x * y = x`（左投影律，所有运算结果等于左操作数）
    结论：`(x * y) * z = x * (y * z)`
    正确答案：True
    解析：根据前提，左边`(x*y)*z = x*z = x`；右边`x*(y*z) = x*y = x`，左右两边相等，必然成立（左投影律天然满足结合律）。

5.  前提：`x * x = x` 且 `x * y = y * x`（幂等+交换）
    结论：`(x * y) * (x * y) = x * y`
    正确答案：True
    解析：结论本身就是幂等律的形式，前提已经明确所有元素满足`x*x=x`，将`(x*y)`看作一个整体，必然满足幂等律，因此成立。

---

### False示例（5道，均附带可验证的反例）
1.  前提：`x * x = x`（幂等律）
    结论：`x * y = y * x`（交换律）
    正确答案：False
    解析：构造反例：集合`{0,1}`，定义运算`*`为「左投影」：`x*y = x`。该运算满足幂等律（`x*x=x`），但不满足交换律：`0*1=0`，`1*0=1`，二者不相等。存在满足前提但不满足结论的magma，因此为False。

2.  前提：`x * y = y * x`（交换律）
    结论：`x * (y * z) = (x * y) * z`（结合律）
    正确答案：False
    解析：构造反例：石头剪刀布运算，集合`{0,1,2}`（0=石头，1=剪刀，2=布），`x*y`为获胜方，平局为`x`，运算表对称（满足交换律），但不满足结合律：`0*(1*2)=0*1=0`，`(0*1)*2=0*2=2`，二者不相等，反例成立。

3.  前提：`x * (y * z) = (x * y) * z`（结合律）
    结论：`x * y = y * x`（交换律）
    正确答案：False
    解析：反例：字符串拼接运算，满足结合律（`"a"+("b"+"c")=("a"+"b")+"c"`），但不满足交换律（`"ab"≠"ba"`），因此为False。

4.  前提：`x * x = x`（幂等律）
    结论：`x * (x * y) = y`
    正确答案：False
    解析：反例还是左投影运算`x*y=x`，满足幂等律，但`x*(x*y)=x*x=x≠y`（只要`x≠y`），因此不成立。

5.  前提：`x * y = x`（左投影律）
    结论：`x * y = y * x`（交换律）
    正确答案：False
    解析：左投影运算中，`x*y=x`，`y*x=y`，当`x≠y`时二者不相等，反例成立，因此为False。

## 二、比赛常规难度示例（10道，对应公开训练集常规题）
### True示例（5道）
1.  前提：`x * (y * z) = (x * y) * z`（结合律） 且 `x * x = x`（幂等律）
    结论：`(x * y * x) * (x * y * x) = x * y * x`
    正确答案：True
    解析：结合律允许省略括号，将`(x*y*x)`看作一个整体，前提明确所有元素满足幂等律，因此该整体也满足`a*a=a`，结论成立。

2.  前提：`x * y = y * x`（交换律） 且 `x * (y * z) = (x * y) * z`（结合律）
    结论：`(x * y) * (z * w) = (w * z) * (y * x)`
    正确答案：True
    解析：根据结合律可调整括号不改变结果；根据交换律，`x*y=y*x`、`z*w=w*z`，因此左边`(x*y)*(z*w) = (y*x)*(w*z) = (w*z)*(y*x)`，与右边相等。

3.  前提：`x * (x * y) = x * y`（左吸收律）
    结论：`x * (x * (x * y)) = x * y`
    正确答案：True
    解析：根据前提，`x*(x*y)=x*y`，将结论左边的`x*(x*y)`替换为`x*y`，左边变为`x*(x*y)`，再根据前提替换为`x*y`，与右边相等。

4.  前提：`x * (y * z) = (x * y) * z`（结合律） 且 `x * y = y * x`（交换律）
    结论：`x * (y * z) = z * (y * x)`
    正确答案：True
    解析：根据结合律，左边`x*(y*z) = (x*y)*z`；根据交换律，`(x*y)*z = z*(x*y) = z*(y*x)`，与右边相等。

5.  前提：`x * x = x` 且 `x * (y * z) = (x * y) * z` 且 `x * y = y * x`（幂等+结合+交换）
    结论：`x * (y * z) = (x * y) * (x * z)`（自分配律）
    正确答案：True
    解析：根据幂等律`x=x*x`，左边`x*(y*z) = (x*x)*(y*z)`；再根据结合律与交换律，调整运算顺序为`(x*y)*(x*z)`，与右边完全一致。

---

### False示例（5道）
1.  前提：`x * x = x` 且 `x * (y * z) = (x * y) * z`（幂等+结合）
    结论：`x * y = y * x`（交换律）
    正确答案：False
    解析：反例：幂等矩阵的乘法，满足结合律与幂等律，但矩阵乘法不满足交换律。例如`A=[[1,0],[0,0]]`，`B=[[1,1],[0,0]]`，均为幂等矩阵，`A*B=[[1,1],[0,0]]`，`B*A=[[1,0],[0,0]]`，二者不相等，反例成立。

2.  前提：`x * (y * x) = x`（左吸收律）
    结论：`x * (x * y) = x`
    正确答案：False
    解析：构造反例：集合`{a,b}`，运算表如下：
    | * | a | b |
    |---|---|---|
    | a | a | a |
    | b | b | a |
    验证前提：对任意x,y，`x*(y*x)=x`均成立；但当`x=b,y=a`时，`x*(x*y)=b*(b*a)=b*b=a≠b`，不满足结论，反例成立。

3.  前提：`x * y = y * x` 且 `x * x = x`（交换+幂等）
    结论：`x * (y * z) = (x * y) * z`（结合律）
    正确答案：False
    解析：反例为前文的石头剪刀布运算，满足交换律与幂等律，但`0*(1*2)=0≠2=(0*1)*2`，不满足结合律。

4.  前提：`x * (x * y) = x * y`（左吸收律）
    结论：`(x * y) * y = x * y`（右吸收律）
    正确答案：False
    解析：构造反例：集合`{0,1,2}`，运算表如下：
    | * | 0 | 1 | 2 |
    |---|---|---|---|
    | 0 | 0 | 1 | 0 |
    | 1 | 0 | 1 | 0 |
    | 2 | 0 | 1 | 1 |
    验证前提：对任意x,y，`x*(x*y)=x*y`均成立；但当`x=2,y=2`时，`(x*y)*y=1*2=0≠1=x*y`，不满足结论，反例成立。

5.  前提：`x * (y * z) = (x * y) * z` 且 `x * x = x`（结合+幂等）
    结论：`x * (y * z) = (x * y) * (x * z)`（自分配律）
    正确答案：False
    解析：构造反例：2个生成元的自由带（幂等结合半群），取`x=a,y=b,z=b`，左边`a*(b*b)=a*b`，右边`(a*b)*(a*b)=a*b`，换一个：取`x=a,y=b,z=a`，在非交换自由带中，`a*(b*a)≠(a*b)*(a*a)`，不满足自分配律，反例成立。

## 三、高难度挑战示例（6道，对应比赛hard题）
### True示例（3道）
1.  前提：`x * (y * z) = (x * y) * z` 且 `x * y = y * x` 且 `x * x = x`（结合+交换+幂等）
    结论：`x * (y * (x * z)) = x * (y * z)`
    正确答案：True
    解析：根据结合律与交换律，左边`x*(y*(x*z)) = x*(y*x*z) = x*(x*y*z) = (x*x)*(y*z) = x*(y*z)`，与右边相等。

2.  前提：`(x * y) * x = x`（中吸收律）
    结论：`x * x = x`
    正确答案：True
    解析：将前提中的`y`替换为`x`，得到`(x*x)*x = x`；再将前提中的`y`替换为`x*x`，得到`(x*(x*x))*x = x`；最终可推导出`x*x = (z*x)*x = z*x = x`，满足幂等律。

3.  前提：`x * (y * x) = x` 且 `(x * y) * y = x * y`（左吸收+右吸收）
    结论：`(x * y) * (z * w) = x * w`
    正确答案：True
    解析：同时满足左、右吸收律的magma为矩形带，天然满足`(x*y)*(z*w)=x*w`的核心性质，因此结论成立。

---

### False示例（3道）
1.  前提：`x * (y * x) = x`（左吸收律）
    结论：`(x * y) * x = x`（中吸收律）
    正确答案：False
    解析：构造反例：集合`{0,1}`，运算表如下：
    | * | 0 | 1 |
    |---|---|---|
    | 0 | 0 | 0 |
    | 1 | 1 | 0 |
    验证前提：对任意x,y，`x*(y*x)=x`均成立；但当`x=1,y=0`时，`(x*y)*x=1*1=0≠1=x`，不满足中吸收律，反例成立。

2.  前提：`x * (y * z) = (x * y) * z` 且 `x * x = x` 且 `(x * y) * (y * x) = x * y`
    结论：`x * y = y * x`（交换律）
    正确答案：False
    解析：反例：左投影半群`x*y=x`，满足结合律、幂等律，且`(x*y)*(y*x)=x*y=x=x*y`，符合前提，但不满足交换律。

3.  前提：`x * (y * z) = (x * y) * z`（结合律） 且 `x * x * x = x`（3阶幂等）
    结论：`x * x = x`（幂等律）
    正确答案：False
    解析：构造反例：集合`{0,1}`，异或运算`x*y = (x+y) mod 2`，满足结合律，且`x*x*x = (x+x)+x = 0+x = x`，符合前提，但`x*x=0≠x`（当x=1时），反例成立。
