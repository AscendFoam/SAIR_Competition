- public selected problems
- `equations.txt`
- implication graph exports
- `prompt_templates.jsonl`
- `cheatsheet_balanced.md`
- `cheatsheet_counterexample_first.md`

- `normal.jsonl`
- `hard1.jsonl`
- `hard2.jsonl`
